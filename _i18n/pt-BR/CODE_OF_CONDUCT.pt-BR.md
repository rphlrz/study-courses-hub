# Código de Conduta

## Nosso Compromisso

Comprometemo-nos a manter um ambiente acolhedor e respeitoso para todos que interagem com este repositório.  

Como este projeto é uma coleção pessoal de **Raphael (@rphlrz)**, espera-se que colaboradores e visitantes ajam com integridade, gentileza e profissionalismo.

## Nossos Padrões

Exemplos de comportamento positivo:
- Usar linguagem respeitosa e inclusiva  
- Oferecer feedback construtivo  
- Reconhecer diferentes pontos de vista  
- Focar no aprendizado e na colaboração

Comportamentos inaceitáveis incluem:
- Assédio ou discriminação de qualquer tipo  
- Provocações, insultos ou ataques pessoais  
- Compartilhar ou usar conteúdo sem permissão  
- Promover conteúdo externo ou comercial

## Aplicação

Casos de comportamento inaceitável podem ser relatados diretamente pelo GitHub ou pela seção de contato do repositório.  
Os relatos serão analisados e tratados de forma privada por **Raphael (@rphlrz)**.
