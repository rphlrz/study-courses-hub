# Intermediary-BackEnd

---

This repository contains the code I developed during hands-on classes where I learned how to integrate a Back-End application with a Front-End application, while also building my understanding of concepts such as code versioning and the Node.js framework with its internal and external modules.

I adopted **Git Flow** as the standard development workflow for this project.

# Project: Intermediary-Backend

> In this project I built an integrated Back-End and Front-End application, exploring code versioning, the Node.js framework, and its internal and external modules through practical exercises.

## 💻 Prerequisites

Before running this project, make sure the following requirements are met:

* I used the latest version of `Node.js` — you should have it installed as well.
* I developed this on a `Windows` machine, but it should work on `Linux / Mac` too.
* I recommend watching the environment setup video lesson before getting started.

## 🚀 Running the Project

To start the back-end, navigate to the `lgpd-back` directory and run:

```
node index.js
```

To start the front-end, navigate to the `lgpd-front` directory and run:

```
ng serve
```
